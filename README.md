# Java Fundamentals

Repositório com exercícios práticos de Java desenvolvidos durante o curso de Análise e Desenvolvimento de Sistemas (ADS).

## 🎯 Objetivo

Praticar e consolidar os fundamentos da linguagem Java, com foco em Programação Orientada a Objetos (POO).

## 📚 Conteúdo

### Aula 02 — Classes e Objetos
- Criação de classes com atributos e métodos
- Instanciação de objetos
- Encapsulamento básico
- Exemplo prático: classe `Caneta` com atributos `modelo`, `cor`, `ponta`, `carga` e `tampada`

## 🛠️ Tecnologias
- Java
- Apache NetBeans IDE 28
