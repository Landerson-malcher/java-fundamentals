# Conceitos de POO — Classe Caneta

Exercício prático para consolidar os fundamentos de Programação Orientada a Objetos (POO) em Java.

## O que foi praticado

- Criação de classes com atributos e métodos
- Instanciação de objetos
- Encapsulamento básico com getters e setters
- Exemplo prático: classe Caneta com atributos modelo, cor, ponta, carga e tampada

## Tecnologias

- Java
- Apache NetBeans IDE 28
