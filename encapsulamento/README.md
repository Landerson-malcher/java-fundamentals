# Encapsulamento — Controle Remoto

Exercício prático de encapsulamento em Java utilizando interface, getters, setters e lógica orientada a objetos.

## O que foi praticado

- Encapsulamento com atributos privados
- Getters e setters
- Implementação de interface
- Estruturas condicionais (if, else)
- Laço de repetição (for)
- Exemplo prático: simulação de um controle remoto com métodos ligar, desligar, play, pause, mudo e volume

## Tecnologias

- Java
- Apache NetBeans IDE 28
